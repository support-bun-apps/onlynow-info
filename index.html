OnlyNow プライバシーポリシー

個人開発者BunDev（以下「当方」）は、本アプリケーション「OnlyNow」（以下「本アプリ」）におけるユーザーの情報の取り扱いについて、以下の通りプライバシーポリシーを定めます。

1. 基本方針

当方は、ユーザーのプライバシー保護を最優先事項とし、アプリの利便性向上に必要な最小限のデータのみを扱います。原則として、ユーザーが入力した情報を当方のサーバーに送信・蓄積することはありません。

2. 取得する情報と利用目的

  • タスクおよび履歴データ: アプリ内で入力されたタスク内容や完了履歴は、ユーザーの端末内（Roomデータベース）にのみ保存されます。これらはアプリの機能提供（タスク表示・管理）のために使用されます。
  • 設定データ: タイマー時間や通知間隔、テーマ設定などは端末内（DataStore）に保存されます。
  • 音声入力データ: 音声認識機能を使用する際、Android標準の SpeechRecognizer API を使用します。音声データはテキスト変換のため、Googleのサーバーに送信され処理される場合があります。当方が音声データ自体を収集・記録・二次利用することはありません。

3. 権限の使用について

本アプリは機能提供のため、以下の権限を使用します。取得した情報を第三者に提供することはありません。

  • マイク (RECORD_AUDIO): 音声によるタスク入力機能のため。
  • 通知 (POST_NOTIFICATIONS): タスクのリマインダー通知を送るため。
  • アラーム (SCHEDULE_EXACT_ALARM): 設定時間に正確に通知を作動させるため。

4. 第三者によるデータ収集（決済）

本アプリは、プレミアム機能の提供にあたり「Google Play Billing」を利用しています。決済に関する情報（クレジットカード番号等）はGoogle社によって直接管理され、当方がユーザーの決済詳細情報にアクセスすることはありません。
