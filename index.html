OnlyNow プライバシーポリシー
Privacy Policy

本プライバシーポリシーは、スマートフォン向けアプリ「OnlyNow」（以下、「本アプリ」）におけるユーザー情報の取り扱いについて説明するものです。

This privacy policy explains how user information is handled in the smartphone application "OnlyNow".

1. ユーザーデータの収集と利用について / Collection and Use of Data

本アプリは、ユーザーが入力したタスク内容や履歴データを、端末内のローカルデータベース（Room）および設定ファイル（DataStore）にのみ保存します。これらのデータが開発者のサーバーや第三者に送信・収集されることは一切ありません。

This App stores task details and history data entered by the user only in the local database (Room) and preference files (DataStore) on the device. This data is never sent to or collected by the developer's server or any third party.

2. 音声入力について / Audio Input

本アプリは、音声によるタスク登録機能を提供するため、マイクへのアクセス権限を使用します。音声データはAndroid標準の音声認識エンジンによりデバイス上で処理され、録音データの保存や外部への送信は行いません。

This App uses microphone access to provide a voice task registration feature. Audio data is processed on the device by the standard Android speech recognition engine, and no audio recordings are stored or transmitted externally.

3. アプリ内課金について / In-App Billing

プレミアム機能の提供にあたり「Google Play Billing」を使用しています。決済処理はGoogle社によって行われ、開発者がユーザーのクレジットカード情報などの決済情報を直接取得・保持することはありません。

We use "Google Play Billing" to provide premium features. Payment processing is handled by Google, and the developer does not directly obtain or hold any payment information, such as credit card details.

4. バックアップについて / Data Backup

本アプリは、ユーザーの利便性のためにAndroid標準の自動バックアップ機能を利用しますが、セキュリティ上の理由から、プレミアム購入状態を含む一部の設定情報はバックアップ対象から除外されています。

This App uses the standard Android automatic backup feature for user convenience. For security reasons, certain configuration information, including premium purchase status, is excluded from backups.

5. プライバシーポリシーの変更 / Changes

本ポリシーの内容は、法規制の変更や機能追加に伴い、事前の通知なく改定される場合があります。

The contents of this policy may be revised without prior notice due to changes in laws, regulations, or added features.
